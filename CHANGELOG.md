### Breaking change

When migrating from 5.X to 6.X, you have to add the default menu item types to your config file if you wish to continue to use them.

The config would look like this:

```php
'menu_item_types' => [
  \Workup\MenuBuilder\MenuItemTypes\MenuItemTextType::class,
  \Workup\MenuBuilder\MenuItemTypes\MenuItemStaticURLType::class,
  // ...
],
```
